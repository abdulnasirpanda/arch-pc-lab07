## Описание
Лабораторная работа по изучению безусловных и условных переходов в ассемблере NASM.

## Структура проекта
- lab7-1.asm - Программа с безусловными переходами (jmp)
- lab7-2.asm - Программа с условными переходами (cmp + jg) 
- smallest.asm - Программа поиска наименьшего числа
- function.asm - Программа вычисления функции f(x)
- in_out.asm - Библиотека ввода-вывода
- lab7-2.lst - Файл листинга
- report.md - Отчет по лабораторной работе
